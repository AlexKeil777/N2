Project overview

VHP tubs decontamination machine & fast, deep H₂O₂ removal

This project proposes an innovative approach to the decontamination of medical tubs using vaporous hydrogen peroxide (VHP), addressing key technical and operational challenges in pharma/biotech settings where RTU tubs enter aseptic processing.

The core idea is twofold: rethink the process architecture (moving from continuous tunnels to a modular batch concept) and optimize aeration to ensure thorough H₂O₂ removal before sterile transfer.

Batch / modular process Operational reliability Flexible recipes Deep residual reduction
